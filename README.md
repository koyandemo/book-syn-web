## Run Locally

Clone the project

```bash
  git clone https://github.com/koyandemo/book-syn-web
```

Go to the project directory

```bash
  cd book-syn-web
```

Install dependencies

```bash
  yarn install
```

Start the application

```bash
  yarn dev
```
